1. npm init
2. Если угодно, создать папку node_modules. Вроде должна создатья сама, но я делаю на всякий случай.
3. npm install gulp --save-dev
4. Создать файл настроек gulpfile.js
5. Радуемся, все намного проще чем в webpack =)


npm install gulp-stylus --save-dev            // Стилус
npm install autoprefixer-stylus --save-dev

npm install gulp-plumber --save-dev			  // Иногда, когда мы "вотчим" файл, могут возникнуть ошибки и тогда все повалится.
												пламбер перехватывает ошибки и не дает всему упатсть
npm install gulp-concat --save-dev			  // Соединяет файлы в один


В отличие от webpack в gulp нат таких сложных преколов с модульностью. Он просто выполнятет описанные задания(task). 
Эти файлы так то обработать, эти так то и тд.

Чтобы запустить галп необходимо выполнить простую команду gulp.
В таком случае выполнится task с названием default. Если такого в настройках нет, то выведется ошибка.

Галпу нужно понимать, какой таск выполнить:

gulp stylus
gulp jsconcat

или 

gulp watch

который "следит" за первыми двумя.
